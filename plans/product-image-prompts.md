# Промты для генерации изображений IT-услуг

> Дата: 2026-05-14
> Источник: `plans/product-catalog-plan.md`, раздел C

---

## Общие параметры генерации

| Параметр | Значение |
|----------|----------|
| Размер | 1024×1024 |
| Стиль | Isometric / flat professional illustration |
| Палитра | Teal, navy, warm beige/cream, muted neutrals |
| Эстетика | Деловой, корпоративный, минималистичный |
| Текст на изображении | Нет (no text, no logos) |
| Формат файла | PNG |
| Совместимость | Midjourney, DALL-E 3, Stable Diffusion XL |

### Рекомендации по генерации

- Добавляйте `--ar 1:1` для Midjourney
- Для DALL-E 3: используйте промт как есть, укажите размер 1024x1024
- Для Stable Diffusion: добавьте negative prompt: `text, letters, words, watermark, logo, blurry, low quality, photorealistic`
- Все промты оптимизированы под единый визуальный стиль каталога
